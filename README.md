# 算術鬧鐘 (Arithmetic Alarm Clock)

基於香港數學課程的全前端鬧鐘應用。設定鬧鐘後，觸發時必須解開對應年級的算術題才能關閉鈴聲。

## 檔案結構

```
calculate/
├── index.html    # 主頁面，包含三個畫面的 HTML 骨架
├── styles.css    # 所有樣式：響應式佈局、動畫、元件外觀
└── app.js        # 所有邏輯：資料層、鬧鐘排程、算術題生成、音效
```

## 畫面說明

### 1. 鬧鐘列表主頁 (`#screen-main`)
- 已建立鬧鐘卡片，依觸發時間排序
- 每張卡片顯示：時間、重複規則、年級、標籤、啟用開關
- 長按卡片滑出刪除按鈕，點擊進入編輯
- 空白狀態插圖提示，右下角 FAB「+」按鈕

### 2. 鬧鐘設定頁 (`#screen-settings`)
- 時間選擇（時/分下拉選單）
- 重複模式：單次 / 每天 / 自訂（勾選星期一至日）
- 香港數學年級選擇（小一至中六 + DSE 選修 M1/M2）
- 題目數量選擇（1-10 題，需全部答對才能關閉鬧鐘）
- 自訂標籤文字框
- 編輯模式自動帶入現有數值

### 3. 算術挑戰頁 (`#screen-challenge`)
- 鬧鐘觸發時強制全屏，無法返回或關閉
- Web Audio API 循環播放 880Hz/440Hz 交替嗶聲
- 根據年級動態生成算術題（全部可心算）
- 4 欄數字鍵盤（含分數 `/` 及小數點 `.`）+ 實體鍵盤輸入
- 多題模式：顯示進度「第 2/5 題」，每題正確自動切換下一題
- 全部答對：停止鈴聲 → 成功動畫 → 2 秒後返回主頁
- 答案錯誤：震動動畫 + 錯誤提示，鈴聲持續，不扣進度
- 「換一題」按鈕可在不停止鈴聲下更換同級題目（不影響進度）

## 香港數學課程題型（按年級）

| 年級 | 題型範圍 |
|------|---------|
| 小一 | 20 以內加減 |
| 小二 | 兩位數加減 |
| 小三 | 乘數表（9×9）及簡單除法 |
| 小四 | 兩位數 × 一位數、除法 |
| 小五 | 異分母分數加減 |
| 小六 | 小數乘除、分數乘除混合 |
| 中一 | 負數四則運算 |
| 中二 | 代數求值（含二次式） |
| 中三 | 一元一次方程 |
| 中四 | 二次方程根、指數定律、聯立方程、簡易根式 |
| 中五 | 對數、排列組合（nPr/nCr）、簡易三角值 |
| 中六 | 等差/等比數列、統計（平均/中位數）、不等式 |
| M1 | 簡易微分（多項式）、定積分 |
| M2 | 2×2 行列式、極限、複數運算 |

## 資料儲存

所有鬧鐘設定儲存於 `localStorage`，key 為 `arithmetic-alarms`。

鬧鐘物件結構：

```js
{
  id: "uuid",           // 唯一識別
  hour: 7,              // 0-23
  minute: 30,           // 0-59
  repeat: "daily",      // "once" | "daily" | "custom"
  repeatDays: [false, true, ...],  // [日,一,二,三,四,五,六], custom 時使用
  grade: "p3",          // "p1"..."p6", "s1"..."s6", "m1", "m2"
  questionCount: 3,     // 1-10，需答對題數才關閉鬧鐘
  label: "起床上學",     // 自訂標籤
  enabled: true,        // 啟用狀態
  nextTrigger: 1715200000000,  // 下次觸發 timestamp
  createdAt: 1715200000000
}
```

## 防逃逸機制（挑戰頁）

- `history.pushState` 雙重推送 + `popstate` 攔截，阻止返回鍵
- `hashchange` 攔截
- `beforeunload` 關閉警告
- `requestFullscreen` 全屏請求
- `touchmove` 手勢阻止（防止手機滑動返回）

## 背景鬧鐘檢查

- `setInterval` 每秒檢查所有鬧鐘的 `nextTrigger` 是否到期
- `visibilitychange` 事件：頁面從背景恢復時立即檢查，補捉錯過的鬧鐘
- Notification API：請求通知權限，背景時推送系統通知

## 語言切換

點擊主頁右上角「中/EN」按鈕即可切換繁體中文與英文介面，所有靜態文字、數學題目說明、年級選項均會即時轉換。語言偏好儲存於 `localStorage`。

## 使用方式

直接在瀏覽器中打開 `index.html` 即可，無需伺服器或建置工具。
